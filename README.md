# **kakkoteemu-react**

My personal website using React.

## **Used technologies**
* React 17.0.2
* Webpack 5.4.1
* [@emotion/styled 11.3.0](https://www.npmjs.com/package/@emotion/styled)

## **npm start**

## **npm run build**
